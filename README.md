# Flying-B-MA
광운대학교 소프트웨어학부 소모임 PU에서 진행한 학술제에서 발표한 작품입니다.



![날아라 비마 이미지](img1.PNG)
비마(말)가(이) 언덕 위에서 뛰면 거리에 비례한 점수와 골드를 얻습니다.


![상점 이미지](img2.PNG)
획득한 골드를 이용해 아이템을 사거나 능력치를 증가시킬 수 있습니다.  (30초 광고를 보면 3000골드를 얻을 수 있습니다.)
